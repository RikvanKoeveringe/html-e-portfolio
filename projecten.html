Calculator: Ik heb een calculator gemaakt met java die plus, min, keer, delen, wortel, kwadraad en andere machten kan berekenen. Bij elke functie kreeg ik een uitleg van wat ik ongeveer zou moeten doen. Ik heb vooral geleerd hoe je verschillende loops en cases kan maken en gebruiken in Java.
